Fujitsu Pricing Tool

This model consists of an online database which is linked to a light-weight web based front end. Once a deal starts, the Q-model can be accessed to assess the target price of the pursuit. The data used relies on benchmark data, past deal data and competitive data. And if a deal is submitted the Q-model is used to normalize and submit the data sent to the client.
